Explicação Bootstrap

O que é bootstrap?

É um framework do CSS. Framework é nada mais que uma caixa de ferramentas pronta para ser usada.

    Pontos positivos:

  • Não precisa de um design pronto para usar.
  • Não precisa escrever uma linha de CSS se não quiser.

    • Pontos negativos:

  • Perdemos a identidade única do site
  • Perde a flexibilidade do CSS, visto que é uma caixa de ferramentas pronta.
