Explicação de Função

Temos que ter uma tag que abre e fecha para funcionar o JS mesmo que ela não tenha conteudo nem um dentro dela. A tag é

A function só é executada quando chamamos ela.
Ex:

  • let idade = 32; (declarando uma variavel)
  • function maisVelho () {
    idade = idade + 1; (Adicionando mais 1 a idade)
    alert (idade); (mostra idade na tela)
    }

Para chamar a function devemos fazer o seguinte, maisVelho(); É necessario o () para chamar a function.

Como chamar a função dentro do botão:

Para atribuirmos o button na variavel botao e depois chama-la:

  • const botao = window.document.querySelector("button");
  • Devemos criar uma variavel. (const no caso)
  • Atribuir o nome dela (botao)
  • Entrar no navegador (.window), para entrar é (.)
  • Entrar no HTML (.document)
  • Fazer a busca no html (.querySelector("button"))
  • No caso usamos o Selector pois estamos buscando um elemento e não classe ou ID
  • Todo elemento dentro do HTML é string, então colocamos tudo dentro de ""

Para chamar a variabel botao ao clicar no button:

  • botao.onclick = ficaMaisVelho;
  • Estamos querendo clicar e chamar a funcao, então começamos com a variavel criado para isso (botao)
  • Para procurar algo na variavel (.onclick), estamos atribuindo o valor onclick ao botao e assim o evento de clicar será efetuado.
  • Atribuimos (=) ficaMaisVelho ao onclick que sera disparado ao clicar no botao.
